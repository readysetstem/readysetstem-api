Class LEDSprite

source code

object --+
         |
        LEDSprite

Allows the creation of a LED Sprite that is defined in a text file.

  • The text file must only contain hex numbers 0-9, a-f, A-F, or - (dash)
  • The hex number indicates pixel color and "-" indicates a transparent pixel
