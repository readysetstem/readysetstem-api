Button Cell

Need:

  • 2 tactile button
  • 4 jumper wires

Setting up the buttons:

  1. Place the 2 tactile buttons onto the breadboard so that horizontally so that one is to the left of the other.
  2. Using a jumper wire, connect one side of each of the buttons to a GND port on the Raspberry Pi.
  3. Using one of the jumper wires, connect the other side of the left button to port 23 on the Raspberry Pi.
  4. Using the other jumper wire, connect the other side of the right button to port 18 on the Raspberry Pi.
