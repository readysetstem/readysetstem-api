Hooking up a button

Need:

  • 1 tactile button
  • 2 jumper wires

Steps:

  1. Connect the tactile button to the breadboard
  2. Using a jumper wire, connect one side of the button to a GND port on the Raspberry Pi
  3. Using the other jumper wire, connect the other side of the button to the specific port number you would like to use.

    These port numbers can be any port that has a number. (ie, anything but 3.3V, 5V, and GND ports)
