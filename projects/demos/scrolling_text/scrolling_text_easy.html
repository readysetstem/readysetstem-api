Programming

Displaying the Text

  1. First, we must draw the text on the LED matrix display. To do this we call led_matrix.text() with the text we want to display between the parenthesis.

    This will tell the program that you want the given text to be shown the next time led_matrix.show() is called.

  2. To actually see our changes to the LED matrix displayed, we next need to call led_matrix.show()

    When displayed we will see part of the text displayed on the matrix. However, not all the text will be displayed.
