Scrolling Text - Medium
Hello World

In this demo we will be showing how to display text on the LED matrix.

To overcome the limitation of the size of the LED matrix display. We will be displaying part of the text at a time by changing the position of the displayed text

Setup

Need:

  • 2 LED matrices connected together to make a 1x2 display

Programming

  1. First, we want to create a while loop that will continuously loop infinitly.

    We can do this by creating the conditional statement to always be True.

  2. In the while loop we first want to clear the previous text that is displayed on the LED matrices. (There will be something displayed after the first iteration through the matrix.)
  3. Next, we display the first part of "Hello World", by drawing the text at position (0,0). Meaning the text will be positioned at the very bottom left of the matrix.
  4. Next, we want to show the text we drew by calling led_matrix.show()
  5. We then want the program to wait a second before starting to display the next part. If we don't do this, the text will be displayed to fast for use to see and read it. By calling time.sleep(1), the program will wait 1 second before moving onto the next line.
