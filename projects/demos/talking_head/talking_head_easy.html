Talking Head - Easy
Hello World

In this demo we will be showing how to get the Raspberry Pi to convert given text such as "Hello World" into speach and then play it.

Setup

Need:

  • 1 LED matrix

Steps:

  1. Connect the speaker??

Programming

  1. First, create led_matrix.LEDSprite() variables mouth_open and mouth_closed to be later used in the upcoming while loop.
  2. Then, we create a speaker.Speech() object variable called my_speech for the text we want the hear. This object converts the text into and object that Raspberry Pi can play.
  3. Then, we play the speech by calling my_speech.play()
  4. Now, we create a while loop that will continuously loop until the speech that we just started playing has finished playing. This loop will continuously open and close the mouth while the speech is playing by displaying the open and closed mouth sprites in succession.
    1. In the while loop we first clear the LED matrix display from the previous sprite that may be displayed.
    2. Next, we first draw the open mouth sprite using led_matrix.sprite()
    3. Then, we want display the sprite that we have just drawn using led_matrix.show().
    4. Then, to make sure the program waits a little bit to allow us to see the sprite before it moves on, we can use time.sleep(.1) to make the program delay for a fraction of a second.
    5. Finally, we want to repeat the last 4 steps to display the sprite of the mouth closed.
