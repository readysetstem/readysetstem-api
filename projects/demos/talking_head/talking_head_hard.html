Talking Head - Hard
Soundboard

In this demo we will be showing how to create a soundboard for the talking head created in medium.

Setup

Need:

  • 1 LED matrix
  • 8 buttons
  • 17 jumper wires

Steps:

  1. Connect the speaker????
  2. Connect led matrix together by ....... TODO: link to already existing instructions?
  3. Place each button in a seperate area of the breadboard
  4. Connect one side of each button to one of the following ports: 4, 17, 25, 24, 23, 18, 27, and 22

    Make sure each button is connected to its own port.

  5. Connect the other side of each button to the negative rail.
  6. Connect the negative rail to a GND port on the Raspberry Pi.

Programming

  1. First we create a dictionary that will match our buttons to speeches that will be talked when the button is pressed.

    The button is created using a Button object from button.Button() and the speech is created by speaker.Speech()

  2. Then, we need to create led_matrix.LEDSprite() object variables for our sprites of a face with its mouth open or mouth close.
  3. Next we create a while loop that will loop infinitly by setting the conditional to True
  4. Next we want to pick out which speech to play based on which button is pressed.

    To do this, we first set our my_speech variable to None and then we go through a while loop until the variable has finally been set.

  5. In this while loop, we use a for loop and an if statement to check to go through each button and see if it is pressed. If it is, we set the given Speech object of that button to the my_speech variable, which will allow the program to exit the while loop.
  6. After we have selected the correct Speech object and set it to my_speech, we then start playing the speech by calling my_speech.play()
  7. Now, we create a while loop that will continuously loop until the speech that we just started playing has finished playing. This loop will continuously open and close the mouth while the speech is playing by displaying the open and closed mouth sprites in succession.
  8. In the while loop we first clear the LED matrix display from the previous sprite that may be displayed.
  9. Next, we first draw the open mouth sprite using led_matrix.sprite()
  10. Then, we want display the sprite that we have just drawn using led_matrix.show().
  11. Then, to make sure the program waits a little bit to allow us to see the sprite before it moves on, we can use time.sleep(.1) to make the program delay for a fraction of a second.
  12. Finally, we want to repeat the last 4 steps to display the sprite of the mouth closed.
